# 📱 Week 1 – Mobile Store (Web Technologies Lab)

## Overview

This project is the **Week 1** lab assignment for the **Web Technologies** course. It is a **static commercial web page** that showcases a mobile store and demonstrates the use of basic HTML elements.

## Features

* 📋 Displays mobile phone details using **HTML tables (`<table>`)**.
* 🖼️ Shows mobile phone images using the **`<img>` tag**.
* 📱 Includes phone specifications in a well-structured tabular format.
* 🎥 Provides product videos and video links using the **`<a>` (anchor) tag**.
* 🌐 Demonstrates the fundamentals of creating a simple commercial website using HTML.

## HTML Concepts Used

* HTML Document Structure
* Headings (`<h1>` – `<h6>`)
* Tables (`<table>`, `<tr>`, `<th>`, `<td>`)
* Images (`<img>`)
* Hyperlinks (`<a>`)
* Basic Page Formatting

## Learning Outcomes

Through this project, I learned how to:

* Create a static web page using HTML.
* Organize data using tables.
* Embed and display images.
* Add hyperlinks to external resources and videos.
* Build a simple product catalog for a commercial website.
